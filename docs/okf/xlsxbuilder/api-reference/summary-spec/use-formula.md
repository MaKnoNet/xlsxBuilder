## `boolean useFormula()` (automatisch generierter Record-Accessor)


Keine Parameter. Rueckgabewert: `true` = Summenzeile als Excel-Formel `=SUM(...)`, `false` =
vorberechneter Wert. Primitiv, nie `null`. Keine Exceptions.
